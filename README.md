# tinyboot

tinyboot is a tiny bootloader, trying to fit into a single CD sector, while providing as much features as possible.

## Features

* [ ] Filesystems
  - [x] tinyfs
  - [x] FAT32(ONLY IN ROOT DIR.)
  - [ ] ISO 9660
  - [ ] echfs
* [ ] Executable formats
  - [ ] MEF(32 bits)
  - [ ] ELF(32/64 bits)
  - [ ] Flat binary(16/32/64 bits)
* [ ] Protocols
  - [ ] stivale2
  - [ ] tinyboot_protocol
* [x] Nice menu
* [ ] Long mode
* [ ] SMP
* [ ] VESA
* [ ] Memory maps
